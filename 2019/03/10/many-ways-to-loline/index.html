使用数据的几种方式

RESTful API

这种方式在上一篇文章中已经使用过。

在使用Ajax跨域请求json的方式下，遇到了两点麻烦：

  1. 阿里云服务器上运行的python脚本有不知名的问题。
  2. GitHub Pages的博客上需要使用HTTPS来跨域请求，这就需要一个SSl证书，但是自己的证书不会被浏览器信任，而去阿里云申请证书（有免费的）需要注册域名，太麻烦了，所以就改变一下请求json的方式，改用本地加载数据的方式。
